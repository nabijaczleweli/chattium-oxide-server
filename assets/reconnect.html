What is {logo_s}?
{logo_s} (read: chattium oxide, as in: chemical compound) is a lightweight chat platform written in Rust with an aim for simplicity. Both the client and the server reside on GitHub.

How do I connect to a {logo_s} server?
Just type in the exact URL of this page () into the client, as this document is sent by the {logo_s} server itself.
Latest Windows and Ubuntu client binaries can be downloaded from the latest {logo_s} client release page.
